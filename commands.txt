# Useful Commands

## Docker Commands
docker-compose up -d                    # Start test database
docker-compose down                     # Stop database
docker-compose down -v                  # Remove database and data
docker ps                               # Check running containers
docker logs postgres-agent-test         # View database logs

## Agent Commands
python agent.py "Show all customers"
python agent.py --interactive
python agent.py "What were total sales last month?"
python agent.py "Show top 10 products by sales"
python agent.py "List all customers from Mumbai"
python agent.py "How many orders are in draft state?"

## Database Commands
docker exec -it postgres-agent-test psql -U odoo_readonly -d odoo_test
docker exec -it postgres-agent-test psql -U postgres -d odoo_test
